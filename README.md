# 2303_School

## Aufgaben:

Implementiere jeweils eine Klasse
- ```Student```
- ```Teacher```

sowie ein dazugehöriges Testprogramm

Die beiden Klassen sollen folgende Punkte erfüllen:
- mind. 4 Attribute (Datenkapselung!)
- mehrere Konstruktoren (Überladen) mit unterschiedlichen Parametern
- Getter/Setter-Methoden
- mind. 2 Methoden pro Klasse die etwas spezielles machen (z.B. prüfen,...)
- toString für formatierte(!!) Ausgabe


Erstelle ein Testprogramm für die Aufgabenstellung ```SchoolTest```
- erzeuge insg. mind. 4 Objekte
- ändere die Zustände der Objekte durch Aufruf der Methoden
- gib die Objekte vorher und nachher inkl. erklärendem Text aus


## Achtung
Kommentiere dein Programm!
